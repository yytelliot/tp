**OnlyTutors is a desktop app for private tutors in Singapore to manage students, lessons, and payments.** While it has a GUI, most of the user interactions happen using a CLI (Command Line Interface).

* If you are interested in using OnlyTutors, head over to the [_Quick Start_ section of the **User Guide**](UserGuide.html#quick-start).
* If you are interested about developing OnlyTutors, the [**Developer Guide**](DeveloperGuide.html) is a good place to start.
